與lex結合

透過在yacc的%union定義數值，之後可以透過lex在分詞的時候定義

%union {
  int intVar;
  char *str;
}

此時可以保存整數

yylval.intVar = atoi(yytext);

字串的話採用

yylval.str = strdup(yytext);
