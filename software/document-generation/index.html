doxygen

安裝

sudo pacman -S doxygen

產生配置文件

doxygen -g config.conf

根據配置文件產生文檔

doxygen config.conf

檢視文檔

chromium html/index.html

配置

編輯配置檔

vim config.conf

為了檢視 Call Graph

EXTRACT_ALL            = YES
HAVE_DOT               = YES
CALL_GRAPH             = YES
CALLER_GRAPH           = YES
