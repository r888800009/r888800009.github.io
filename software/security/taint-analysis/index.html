安裝工具

透過 aur 安裝並且編譯即可

pacaur -S triton-git

如果要讀取 elf 可能需要安裝 lief

pacaur -S lief

Triton

使用重點

  • hooking PLT
  • 標記 tainted
    • ctx.taintMemory()
  • 檢查 tainted
    • ctx.isMemoryTainted()
