rofi

  • Shift + RightorShift + Left切換rofi的模式

透過rofi連上ssh

rofi有兩種模式適用於ssh，分別是ssh與combi，前者僅會有 ssh連線相關選項，後者則是通用的切換選項。

rofi -show ssh 
rofi -show combi

平鋪式視窗管理器

  • bspwm
    • 如果切換到 macos 的使用者，也有採用類似概念的 yabai wm
  • i3-wm
  • qtile，可以透過python編寫配置
  • awesome wm，可以透過lua編寫配置

bspwm 指令

交換桌面

bspc desktop --swap \^1

移除桌面

bspc desktop \^6 --remove

重新命名桌面

bspc desktop focused --rename newname

建立桌面

bspc monitor focused --add-desktops 3321

將桌面像前後移動

bspc desktop focused --swap prev

取得當前名稱

bspc query -D -d focused --names

xdg-open

配置 pdf 預設啟動程式

查詢 pdf 開啟程式

xdg-mime query default application/pdf

設定成 evince

xdg-mime default org.gnome.Evince.desktop application/pdf

無法設定請參考以下指令進行除錯

XDG_UTILS_DEBUG_LEVEL=2 xdg-mime query default application/pdf
vim .config/mimeapps.list

參見

  • Can’t stop xdg-open from using Mendeley for PDFs
  • Problems with using xdg-mime default. The default was successfully set but it is not used via xdg-open

編輯 PDF metadata

安裝

檢視 metadata

exiftool file.pdf

修改 pdf 標題
