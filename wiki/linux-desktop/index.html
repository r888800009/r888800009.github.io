Sat, 2019 Nov 9

選單工具

  • dmenu
  • rofi視窗切換器

rofi

  • Shift + RightorShift + Left切換rofi的模式

透過rofi連上ssh

rofi有兩種模式適用於ssh，分別是ssh與combi，前者僅會有 ssh連線相關選項，後者則是通用的切換選項。

rofi -show ssh 
rofi -show combi

平鋪式視窗管理器

  • bspwm
  • i3-wm
  • qtile，可以透過python編寫配置
  • awesome wm，可以透過lua編寫配置

bspwm 指令

交換桌面

bspc desktop --swap \^1

移除桌面

bspc desktop \^6 --remove

重新命名桌面

bspc desktop focused --rename newname

建立桌面

bspc monitor focused --add-desktops 3321

將桌面像前後移動

bspc desktop focused --swap prev

取得當前名稱

bspc query -D -d focused --names

man網頁輸出

透過瀏覽器檢視man文檔，以html進行排版。以下示範將瀏覽器 設定成chromium並讀取less的文檔。

export BROWSER=chromium 
man -H less
