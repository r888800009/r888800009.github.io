常用快捷鍵

  • Ctrl + m或Ctrl + j 同Enter
  • Ctrl + l 清潔
  • Ctrl + z 暫停當前程式，參見Linux工作管理
  • Ctrl + h 同退回
  • Ctrl + u 移除左邊所有字元，bash移除所有字元
  • Ctrl + w 移除左邊的單字
  • Ctrl + i 同tab
  • Ctrl + d EOF

列出可使用的Shell列表

chsh -l

修改shell

chsh -s /bin/bash

shell 列表

  • bash
  • shell分類

zsh

#切換到emacs鍵綁定模式
set -o emacs

這樣可以兼容bash大部分的鍵綁定

#切換到vi鍵綁定模式
set -o vi

參見vim，預設進入zsh為輸入模式，透過Esc切換到normal mode，透過i回到輸入模式
