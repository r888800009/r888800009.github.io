常用指令

顯示commit的詳細變化

git log --stat

將commit紀錄以可視化的樹狀結構顯示

git log --graph --oneline

合併兩個分支

git merge a b
# 或
git merge b

移除或修改commit，移除使用drop

git rebase -i [SHA]
