常用快捷鍵

  • Ctrl + u 移除所有字元
  • ctrl + l 清理緩衝區
  • Esc, Backspace 左刪一單字
  • Esc, Right 往右跳一個單字
  • Esc, Left 往左跳一個單字
  • Esc, f 下個單字
  • Esc, b 上個單字
  • Ctrl + Right 往右跳一個單字
  • Ctrl + Left 往左跳一個單字

其他快捷鍵

  • Ctrl + o 同Enter
  • Ctrl + / undo
  • Ctrl + k 移除右邊所有字元
  • Ctrl + d 類似delete
  • Esc, d 右刪一單字
  • Ctrl + r 歷史搜尋模式
  • Ctrl + g 離開搜尋模式
  • Ctrl + a 回到開頭
  • Ctrl + e 到尾端
  • Ctrl + f 向右
  • Ctrl + b 向左
  • Alt + b 上個單字
  • Alt + f 下個單字
  • Esc, t交換光標兩側單字
  • Alt + d 移除右邊單字
  • Esc, u 右邊大寫化
  • Esc, l 右邊小寫化
