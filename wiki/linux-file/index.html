Linux文件管理

Thu, 2019 Sep 5

Linux一些常用於文件管理的筆記。

ls

排除指定檔名

ls -I "*.md"

fdupes

找尋重複的檔案

fdupes -d ./

trash

垃圾桶的功能，避免直接rm誤刪檔案。

du

用於統計大小。

只顯示一層

du --max-depth=1

將輸出的大小轉成kb, mb, gb的格式

du . -h

tar

壓縮解壓縮。

常用參數

  • -x 解壓縮
  • -t 查看內容
  • -c 壓縮
  • -f 壓縮檔名

列出內容

列出file.tar裡面的內容

tar -tf file.tar

find

搜尋該目錄含有linux的文件

find . -exec grep -H "linux" {} \;

rsync

同步工具

rsync 參數 來源 目標 
rsync -avh test1/ test2/

file

用於識別檔案類型

file ./a.out

diff

比較兩個檔案

diff file1 file2

比較兩個資料夾

diff --brief -r dir1/ dir2/

sort

多種排序功能，可依字典序、數字大小等

數字大小排序

sort file --numeric-sort

按照字典序

sort file

stat

顯示文件的詳細資訊，如日期、權限等相關資訊

stat file
