重音符號

重音符號放置

``test`test``

輸出結果為test`test

在code block內放置code block

​```` plaintext
​``` plaintext
```
````

輸出結果為

​``` plaintext
```

多個重音符號 ​```` plaintext ​test``test


輸出結果為
```test``test```

## 嵌入HTML
kbd 標籤，部份markdown渲染會產生顯示結果(CSS)，如github中可用

使用方法如下
``` html
<kbd>ctrl</kbd> + <kbd>c</kbd>
```

hugo因為安全性，採用html標籤時必須進行相關設定才會顯示。
