可容錯輸入部份目錄批配，並且進入目錄

安裝

pacaur -S autojump

在 .zshrc 添加

source /etc/profile.d/autojump.sh

指令

  • j 跳到目錄包含
  • jc 跳到子目錄
  • jo 打開 file manager 跳到目錄
  • jco 打開 file manager 跳到子目錄

檢視手冊

man autojump  
j -h

清理不存在的目錄紀錄

autojump --purge

Ranger autojump

ranger-autojump

複製到 ranger 的 plugins

wget https://raw.githubusercontent.com/fdw/ranger-autojump/master/autojump.py \
  -P ${XDG_CONFIG_HOME}/ranger/plugins/
