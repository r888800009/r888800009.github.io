Pacman

顯示本地包

pacman -Qqm

重新安裝一切

pacman -Syu $(pacman -Qnq)

清理快取

sudo paccache -r

pacaur

安裝pacaur或更新時可用以下腳本。

cd /tmp
git clone https://aur.archlinux.org/pacaur.git
cd pacaur
makepkg -is
cd ..
git clone https://aur.archlinux.org/auracle-git.git
cd auracle-git
makepkg -is
