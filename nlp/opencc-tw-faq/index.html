繁體中文與臺灣正體的差異

“麵包” 輸出成 “麪包”

>>> from opencc import OpenCC
>>> cc = OpenCC('s2t')
>>> cc.convert('面包')
'麪包'
>>> cc = OpenCC('s2tw')
>>> cc.convert('面包')
'麵包'

可以發現使用 s2tw 才能輸出 “麵包”
